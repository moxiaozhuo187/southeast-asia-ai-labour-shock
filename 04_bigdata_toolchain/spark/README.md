## Hive Validation Files

- `04b_create_hive_tables.sql` — run in Hive CLI to create external tables
- `04b_hive_queries_readable.sql` — Hive validation queries, run via Zeppelin `%sql`
- `04b_zeppelin_hive_query_validation.json` — exported Zeppelin note, can be imported directly into Zeppelin

To import the Zeppelin note:
1. Open Zeppelin at `127.0.0.1:9995`
2. Click the import button on the main page
3. Select `04b_zeppelin_hive_query_validation.json`
