

\## Hive Validation Files





\- `04b\_create\_hive\_tables.sql` — run in Hive CLI to create external tables

\- `04b\_hive\_queries\_readable.sql` — Hive validation queries, run via Zeppelin `%sql`

\- `04b\_zeppelin\_hive\_query\_validation.json` — exported Zeppelin note, can be imported directly into Zeppelin



To import the Zeppelin note:

1\. Open Zeppelin at `127.0.0.1:9995`

2\. Click the import button on the main page

3\. Select `04b\_zeppelin\_hive\_query\_validation.json`
